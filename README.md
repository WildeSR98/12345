# Zapret Manager

Программа для обхода блокировок Discord, YouTube и других сайтов на Windows.

> **Важно**: Discord работает, а YouTube — нет? Включите **Secure DNS** в браузере:
> - **Chrome/Edge**: Настройки → Конфиденциальность → Безопасность → Использовать безопасный DNS → Google (Public DNS)
> - **Firefox**: Настройки → Параметры сети → DNS через HTTPS → Максимальная защита → Google

## 🚀 Как установить

### Шаг 1. Скачайте
Скачайте [последний релиз](https://github.com/WildeSR98/12345/releases) и распакуйте архив в любую удобную папку (например, `C:\zapret\`).

### Шаг 2. Запустите
Откройте папку `publish` и запустите **`zapret-manager.exe`** от имени администратора:
- Правой кнопкой мыши → **«Запуск от имени администратора»**

### Шаг 3. Следуйте подсказкам
Менеджер сам всё сделает:
1. Проверит обновления менеджера и ядра
2. Покажет **главное меню** — выберите нужное действие
3. Если выбрали установку — протестирует стратегии и установит лучшую
4. Готово! YouTube, Discord и другие сайты должны заработать

> **Папку можно перемещать** — менеджер работает из любого расположения

## 🏠 Главное меню (при запуске)

При запуске `zapret-manager.exe` вы увидите главное меню:

```
  ══════════════════════════════════════════════════════
  ГЛАВНОЕ МЕНЮ
  ══════════════════════════════════════════════════════
    [1]  Установить / Обновить конфигурацию
    [2]  Удалить zapret
    [3]  Переустановить
    [4]  Диагностика и отчёт
    [5]  Тест стратегий и установка
    [s]  Сервисное меню
```

| Пункт | Что делает | Когда нужен |
|-------|-----------|-------------|
| **[1] Установить / Обновить** | Основной режим. Проверяет обновления, скачивает свежие списки, предлагает выбрать способ установки: автоматический тест, DPI тест или ручной выбор стратегии. Затем устанавливает службу Windows | **Первый запуск** или когда нужно обновить стратегию |
| **[2] Удалить zapret** | Останавливает и удаляет все службы zapret с компьютера | Если хотите полностью отключить обход блокировок |
| **[3] Переустановить** | Сначала удаляет старую службу, потом устанавливает заново | Если что-то сломалось и нужно начать с нуля |
| **[4] Диагностика и отчёт** | Проверяет доступность Discord, YouTube и других сайтов, затем сохраняет отчёт в файл | Если нужно понять что работает, а что нет |
| **[5] Тест стратегий** | Тестирует все стратегии и показывает какая лучше для вашего провайдера. Предлагает установить лучшую | Если текущая стратегия перестала работать |
| **[s] Сервисное меню** | Открывает расширенное меню управления (описано ниже) | Для тонкой настройки |

### Что происходит при выборе [1]:

1. 📥 **Проверка обновлений** — менеджер проверяет есть ли новая версия себя и ядра zapret
2. 📋 **Обновление списков** — скачивает свежие hosts и IPSet с оригинального репозитория
3. 🔧 **Выбор способа установки**:
   - **[1] Автоматический тест** — тестирует стратегии (стандартный или DPI режим) и ставит лучшую
   - **[2] Ручной выбор** — показывает список стратегий, вы сами выбираете номер
4. ✅ **Установка** — выбранная стратегия устанавливается как служба Windows
5. 🔍 **Проверка** — после установки проверяет доступность сайтов

## 🔧 Сервисное меню (расширенное)

Для доступа: из главного меню нажмите **[s]**, или запустите `zapret-manager.exe --menu`.

### Служба

| Пункт | Что делает | Когда нужен |
|-------|-----------|-------------|
| **1. Установить службу** | Показывает список стратегий, вы выбираете номер — служба устанавливается | Если хотите сменить стратегию или переустановить |
| **2. Удалить службы** | Полностью удаляет все службы zapret | Если хотите отключить обход блокировок |
| **3. Проверить статус** | Показывает работает ли служба zapret | Для проверки что всё ОК |

### Настройки

| Пункт | Что делает | Когда нужен |
|-------|-----------|-------------|
| **4. Игровой фильтр** | Включает/выключает фильтрацию игрового трафика (TCP/UDP порты). Если включён — обход работает только для нужных сайтов, игры не затрагиваются | Если лагают онлайн-игры |
| **5. IPSet фильтр** | Переключает режим фильтрации IP-адресов: «загружен» (только нужные IP), «any» (все IP), «none» (выключен) | Обычно менять не нужно |
| **6. Автопроверка обновлений** | Включает/выключает автоматическую проверку обновлений при каждом запуске | Если хотите отключить проверки |

### Обновления

| Пункт | Что делает | Когда нужен |
|-------|-----------|-------------|
| **7. Обновить IPSet** | Скачивает свежий список IP-адресов. Ваши добавления сохраняются | Если перестали работать какие-то сайты |
| **8. Обновить Hosts** | Скачивает свежий файл hosts с доменами | Если добавились новые заблокированные домены |
| **9. Проверить обновления** | Проверяет есть ли новая версия менеджера или ядра zapret, и предлагает обновить | Раз в неделю или при проблемах |

### Инструменты

| Пункт | Что делает | Когда нужен |
|-------|-----------|-------------|
| **10. Диагностика** | Проверяет доступность Discord, YouTube и других сайтов. Показывает что работает, а что нет | Если что-то не работает — запустите первым делом |
| **11. Тест стратегий** | Тестирует все стратегии и находит лучшую для вашего провайдера (подробнее ниже) | Если текущая стратегия перестала работать |
| **12. Экспорт отчёта** | Сохраняет подробный отчёт о системе в файл | Если нужна помощь — отправьте этот файл |
| **13. TG WS Proxy** | Управление прокси для Telegram | Если нужен прокси для Telegram |

## 🧪 Тестирование стратегий

Доступно при первой установке и через пункт меню **[11]**. Два режима:

### Режим 1 — Стандартный (рекомендуется)

Программа по очереди запускает каждую стратегию и проверяет доступность сайтов:

```
[1/19] general (ALT).bat
──────────────────────────────────────────────────────────
  Discord Main        HTTP:OK    TLS1.2:OK    TLS1.3:OK    | Ping: 42 ms
  YouTube Web         HTTP:OK    TLS1.2:OK    TLS1.3:UNSUP | Ping: 38 ms
  Google DNS 8.8.8.8  Ping: 12 ms
```

- **OK** — сайт доступен ✅
- **ERROR** — сайт недоступен ❌
- **UNSUP** — протокол не поддерживается (не страшно)
- **SSL** — проблема с сертификатом (возможен перехват DNS)

В конце программа покажет какая стратегия набрала больше всего очков и предложит её установить.

### Режим 2 — DPI тест

Продвинутый тест, который проверяет не просто доступность, а **режет ли ваш провайдер трафик**. Отправляет 64KB данных на тестовые серверы и смотрит доходят ли они:

```
  [🇷🇺]Beeline       HTTP:OK    TLS1.2:OK    TLS1.3:UNSUP
  [🇷🇺]MTS           HTTP:OK    TLS1.2:BLOCK TLS1.3:OK    ⚠ 16-20KB freeze
```

- **BLOCK** + **⚠ 16-20KB freeze** — провайдер режет эту стратегию, нужно менять

## ❓ Частые вопросы

### YouTube не работает после установки
1. Запустите **пункт 10 (Диагностика)** — посмотрите что именно не работает
2. Включите **Secure DNS** в браузере (см. начало инструкции)
3. Запустите **пункт 11 (Тест стратегий)** — программа найдёт рабочую стратегию

### Лагают онлайн-игры
Включите **пункт 4 (Игровой фильтр)** — обход будет работать только для нужных сайтов, игровой трафик затрагиваться не будет.

### Перестало работать через время
Провайдеры периодически меняют способы блокировки:
1. **Пункт 9** — проверьте обновления менеджера и ядра
2. **Пункт 7** — обновите список IPSet
3. **Пункт 11** — протестируйте стратегии заново

### Как полностью удалить?
1. Запустите **пункт 2 (Удалить службы)**
2. Удалите папку с программой

## ⚙️ Для продвинутых пользователей

### Параметры командной строки

```
zapret-manager.exe              — мастер автоматической установки
zapret-manager.exe --menu       — интерактивное меню
zapret-manager.exe --remove     — удаление всех служб
zapret-manager.exe --reinstall  — переустановка
zapret-manager.exe --test       — тестирование стратегий
zapret-manager.exe --diagnostics — диагностика
zapret-manager.exe -silent -strategy "FAKE TLS AUTO" — тихая установка
```

### Структура папок

```
📁 publish/                ← Основная папка, запускайте отсюда
│  zapret-manager.exe      ← Менеджер (запускайте это)
│  config.json             ← Настройки
│  
├─ bin/                    ← Ядро zapret (winws.exe)
├─ strategies/             ← Стратегии обхода (general*.bat)
├─ lists/                  ← Списки доменов и IP-адресов
├─ utils/                  ← Служебные файлы
├─ logs/                   ← Логи (создаются автоматически)
└─ backups/                ← Бэкапы (создаются при обновлении)
```

### Сборка из исходников

```bash
cd src/ZapretManager
dotnet publish -c Release -o publish --self-contained false
```

Требуется [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0).
