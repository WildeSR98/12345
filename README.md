# Zapret Autosetup v2.0

Автоматизированный комплекс скриптов для установки, настройки и обновления **Zapret** (ПО для обхода DPI на базе маршрутизации и модификации пакетов) с функцией самообновления.

## Что нового в v2.0

- **Модульная архитектура**: PowerShell-модули (`Utils`, `Update`, `Service`, `Lists`, `Diagnostics`, `Strategy`) вместо монолита 1100+ строк
- **Тихая установка**: `autosetup.bat -silent -strategy "FAKE TLS AUTO"` для автоматизации
- **Транзакционные обновления**: бэкап перед обновлением, автоматический rollback при ошибке
- **Логирование**: структурированные логи в `logs/zapret_YYYY-MM-DD_HH-mm-ss.log`
- **Проверка хэшей**: SHA256 для скачанных архивов
- **Стратегии из JSON**: `strategies.json` + генератор `.bat` вместо 25 копий
- **Параллельные загрузки**: списки качаются параллельно (PowerShell 7+)
- **Диагностические отчёты**: экспорт состояния служб, версий, конфликтов в текстовый файл
- **Поддержка PowerShell 7**: автоопределение `pwsh.exe` с fallback на `powershell.exe`

## Особенности и Функционал

- **Умное Самообновление с GitHub**: Скрипт проверяет свежие коммиты `WildeSR98/12345` и релизы `Flowseal/zapret-discord-youtube`. При обновлении создаётся бэкап; при ошибке — автоматический откат.
- **Интегрированное Тестирование**: Проверка всех стратегий с параллельными потоками (до 8), DPI-чек на 16-20 KB freeze, выбор лучшей конфигурации в 1 клик.
- **Менеджмент Списков (Авто-очистка)**: Merge, дедупликация, валидация CIDR, удаление перекрывающихся подсетей, очистка невалидных IP (`0.x.x.x`). Ваши `*-user.txt` никогда не перезаписываются.
- **Управление Службами**: Корректная остановка WinDivert, удаление конфликтов (GoodbyeDPI и др.), установка как служба Windows.

## Как установить и запустить?

### Интерактивный режим (по умолчанию)

1. Распакуйте архив в удобную папку.
2. Нажмите ПКМ по **`autosetup.bat`** → **«Запуск от имени администратора»**.
3. Следуйте инструкциям в синем окне PowerShell.

**Новое в меню — пункт [5] «Test strategies & install best»:**
- Показывает результаты предыдущих тестов (если есть)
- Тестирует все `general*.bat` стратегии на доступность Discord/YouTube
- Сохраняет историю в `logs/strategy_test_history.json`
- Предлагает установить лучшую стратегию как службу

### Тихий режим (для продвинутных пользователей / автоматизации)

```batch
:: Авто-установка с рекомендуемой стратегией
autosetup.bat -silent

:: Указать конкретную стратегию
autosetup.bat -silent -strategy "FAKE TLS AUTO"

:: Полная переустановка без вопросов
autosetup.bat -silent -mode reinstall -strategy "ALT"

:: Подробное логирование
autosetup.bat -verbose
```

> ⚠️ **Важно**: Никогда не запускайте напрямую `utils/autosetup.ps1`. Всегда используйте `autosetup.bat`. Он гарантирует `ExecutionPolicy Bypass` и права администратора.

## Структура проекта

| Файл/Папка | Назначение |
|------------|------------|
| `autosetup.bat` | Точка входа. Определяет `pwsh`/`powershell`, передаёт аргументы, запрашивает права администратора |
| `utils/autosetup.ps1` | Главный скрипт. Загружает конфиг, импортирует модули, оркестрирует процесс |
| `utils/modules/` | PowerShell-модули с бизнес-логикой |
| `utils/modules/Utils.psm1` | Логирование, бэкап/rollback, прогресс, проверка прав, хэши |
| `utils/modules/Update.psm1` | GitHub API, скачивание с проверкой SHA256, транзакционные обновления |
| `utils/modules/Service.psm1` | Установка/удаление служб zapret, Game Filter, парсинг аргументов winws |
| `utils/modules/Lists.psm1` | Merge списков, очистка, валидация CIDR, параллельное скачивание, hosts |
| `utils/modules/Diagnostics.psm1` | Тесты доступности, DPI-чек, поиск конфликтов, экспорт отчётов |
| `utils/modules/Strategy.psm1` | Загрузка стратегий из JSON, генерация `.bat` файлов |
| `config.json` | Централизованная конфигурация: URL репозиториев, настройки, списки |
| `strategies.json` | Описание стратегий обхода в машиночитаемом виде |
| `service.bat` | Менеджер службы (меню). Пункт 10 — диагностика, пункт 12 — экспорт отчёта |
| `logs/` | Лог-файлы и диагностические отчёты |
| `backups/` | Автоматические бэкапы перед обновлениями (хранится 5 последних) |
